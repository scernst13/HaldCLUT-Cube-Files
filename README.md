# HaldCLUT-Cube-Files
Small Subset of the most used HaldCLUTs I've used but in cube file format form for usage in Real Time LUT application on Lumix Cameras. 

As an avid Darktable user I switched from Fujifilm to Lumix in order to use the actual color profiles I could readily apply in post to my RAW images (specifically for portraits where I found my cosplayers I shoot would want more than what Fujifilm SOOC could offer me). 
I have converted the HaldCLUTs over using LUT Lab.
